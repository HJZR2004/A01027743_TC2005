<a> - Hyperlinks

Sirven para hacer un link de una página a otro recurso en la web, sea otro archivo en el servidor web, u otra página web.

Por ejemplo: si puedes imaginarlo, puedes programarlo

<p> - Paragraph

El tag <p> define un párrafo.

Ejemplo:

Este es un párrafo de ejemplo.

<p>Este es un párrafo de ejemplo.</p>

<h1> - <h6> - Headings

Los tags <h1> a <h6> definen encabezados, donde <h1> es el más importante y <h6> el menos importante.

Ejemplo:

Encabezado 1

Encabezado 2

Encabezado 3

Encabezado 4

Encabezado 5
Encabezado 6
<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>
<h4>Encabezado 4</h4>
<h5>Encabezado 5</h5>
<h6>Encabezado 6</h6>

<div> - Division

El tag <div> define una división o sección en un documento HTML. Se utiliza como un contenedor para otros elementos HTML.

Ejemplo:

Este es un contenedor
<div>Este es un contenedor</div>

<span> - Span

El tag <span> es un contenedor en línea utilizado para agrupar elementos en línea para aplicar estilos o manipular con JavaScript.

Ejemplo:

Texto en línea 
<span>Texto en línea</span>

<strong> - Strong

El tag <strong> define texto fuerte, que generalmente se muestra en negrita.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Molestiae, ex reprehenderit in porro culpa, non exercitationem, itaque repellat sunt provident iusto ab. Inventore quasi provident mollitia unde non commodi laborum. 
<strong>Texto importante</strong>

<i> - Italic

El tag <i> define texto en cursiva.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Blanditiis, beatae cumque rerum iste tempora quisquam laudantium sed placeat! Hic maiores soluta aliquid accusantium esse magni ipsa obcaecati incidunt nihil quibusdam! 
<i>Texto en cursiva</i>

<u> - Underline

El tag <u> define texto subrayado.

Ejemplo:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Ipsam, nam voluptate. Expedita id cum ad sapiente consectetur, dolores vero facere! Dolores ipsam corrupti quia veritatis iusto reiciendis veniam dolore ad. 
<u>Texto subrayado</u>

<b> - Bold

El tag <b> define texto en negrita.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Reprehenderit maiores eveniet dolor suscipit quo magni, molestiae culpa obcaecati perferendis laborum reiciendis, in doloribus ab quia ea dignissimos quasi incidunt vero. 
<b>Texto en negrita</b>

<strike> - Strikethrough

El tag <strike> define texto tachado. Es un elemento obsoleto y se recomienda usar <del> en su lugar.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Qui quasi rem, quaerat fuga numquam nostrum animi libero eos ab optio facilis minima perferendis aliquam at. Amet officia dolorem quas incidunt. 
<strike>Texto tachado</strike>

<small> - Small

El tag <small> define texto pequeño.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Soluta, dolores eos quidem, hic blanditiis repellat cumque ut vero vitae commodi iusto esse iure fuga, id magnam sint ipsum nemo alias. 
<small>Texto pequeño</small>

<em> - Emphasis

El tag <em> define texto enfatizado, que generalmente se muestra en cursiva.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Repudiandae aperiam repellat mollitia totam quae possimus aliquam illo distinctio? Quisquam, cumque corporis. Sed ea earum non quaerat similique quas nulla velit? 
<em>Texto enfatizado</em>

<header> - Header

El tag <header> representa un encabezado de una sección o de una página.

Ejemplo:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Laudantium dolores maiores esse nisi, libero at molestiae beatae consequuntur eius, ipsum magnam totam et ullam quo repellendus? Dolores inventore maiores sed!

<header>
    <h1>Encabezado de la página</h1>
</header>

<section> - Section

El tag <section> define una sección en un documento HTML.

Ejemplo:

Sección de la página

Contenido de la sección

<section>
    <h2>Sección de la página</h2>
    <p>Contenido de la sección</p>
</section>

<article> - Article

El tag <article> define contenido independiente que puede ser distribuido de forma independiente del resto del contenido.

Ejemplo:

Título del artículo

Contenido del artículo

<article>
    <h2>Título del artículo</h2>
    <p>Contenido del artículo</p>
</article>

<aside> - Aside

El tag <aside> define contenido que está al margen del contenido principal.

Ejemplo:

Este es un contenido adicional

<aside>
    <p>Este es un contenido adicional</p>
</aside>

<footer> - Footer

El tag <footer> define el pie de página de una sección o de una página.

Ejemplo:

Pie de página

<footer>
    <p>Pie de página</p>
</footer>

<nav> - Navigation

El tag <nav> define un bloque de enlaces de navegación.

Ejemplo:

  • Inicio
  • Acerca de
<nav>
    <ul>
        <li><a href="#">Inicio</a></li>
        <li><a href="#">Acerca de</a></li>
    </ul>
</nav>

<img> - Image

El tag <img> se utiliza para incrustar una imagen en un documento HTML.

Ejemplo:

<img src="imagen.jpg"

<br> - Line Break

El tag <br> se utiliza para insertar un salto de línea.

Ejemplo:

Este es un texto con un salto de línea.
Este es el texto después del salto de línea.

<br>

<hr> - Horizontal Rule

El tag <hr> define una línea horizontal en la página.

Ejemplo:

<hr>

<form> - Form

El tag <form> se utiliza para crear un formulario HTML para la entrada de usuario.

Ejemplo:

<form>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    <input type="submit" value="Enviar">
</form>

<table> - Table

El tag <table> se utiliza para crear una tabla HTML.

Ejemplo:

titulo titulo
Dato 1 Dato 2 
<table>
    <tr>
        <th>titulo</th>
        <th>titulo</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>

<ul> - Unordered List

El tag <ul> define una lista desordenada.

Ejemplo:

  • Elemento 1
  • Elemento 2
<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>

<ol> - Ordered List

El tag <ol> define una lista ordenada.

Ejemplo:

  1. Elemento 1
  2. Elemento 2
<ol>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ol>
